# Announcement Board

Простий веб-застосунок для роботи з оголошеннями. Користувач може переглядати список оголошень, шукати їх, створювати нові та видаляти.

## Встановлення

1. Клонувати репозиторій:

```bash
git clone https://github.com/EmmaLoi/announcement-board.git
cd announcement-board
```

2. Встановити залежності:

```bash
npm install
```

3. Запустити міграцію бази даних:

```bash
npx prisma migrate dev
```

4. Запустити сервер:

```bash
npm start
```

Після цього застосунок буде доступний за адресою:

```text
http://localhost:3000
```

## Основний функціонал

- Перегляд списку оголошень
- Пошук по назві
- Сортування за датою
- Пагінація по 10 оголошень на сторінку
- Перегляд окремого оголошення
- Створення нового оголошення з валідацією
- Видалення оголошення

## База даних

Використовується SQLite через Prisma.

Модель Announcement містить такі поля:

- id
- title
- description
- price
- category
- contactInfo
- createdAt

## Маршрути

- GET / — головна сторінка зі списком оголошень
- GET /announcements — форма створення оголошення
- POST /announcements — створення оголошення
- GET /announcements/:id — перегляд одного оголошення
- DELETE /announcements/:id — видалення оголошення

## Валідація

Форма створення оголошення має клієнтську та серверну валідацію.

Перевіряються:

- обов'язкові поля
- мінімальна довжина назви
- мінімальна довжина опису
- мінімальна довжина контактної інформації
- допустима категорія
- додатне значення ціни

Якщо форма заповнена неправильно, користувач бачить повідомлення про помилки, а введені дані залишаються у полях.

## Видалення

Видалення оголошення виконується через JavaScript fetch з HTTP методом DELETE. Перед видаленням користувач підтверджує дію через confirm.
